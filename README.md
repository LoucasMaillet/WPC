# Installation

With pip:

```
pip install webpage-compressor
```

# Help

```sh
usage:
wpc [-h] [-n webpage] [-b [mode]] [filePath]

positional arguments:
  filePath              Path of main file.

optional arguments:
  -h, --help            show this help message and exit
  -n webpage, --new webpage
                        Generate new WPC environment..
  -b [mode], --build [mode]
                        Build webpage.
```
